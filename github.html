Git / Github

Så hva er forskjellen mellom Git og GitHub? Git er et system som hjelper deg med å holde orden på flere versjoner av en nettside. Hver gang du endrer noe på siden din vil Git lagre den gamle versjonen av nettsiden som et snapshot. Du kan deretter se alle snapshotene av nettsiden. Hvor mange det er og i hvilken rekkefølge de ble laget i. Du kan bruke denne funksjonaliteten til å eksperimentere og gå tilbake til tidligere versjoner hvis du skulle ønske det. GitHub er en nettside der du kan publisere Git-depotene dine og samarbeide med andre mennesker. Git lagrer filene lokalt på maskinen og du trenger derfor ikke GitHub for å bruke Git.
